# GuidanceSpace Portfolio
A minimalist, scrollable portfolio built with React, Vite, and Firebase Hosting.

## 🚀 Purpose
Showcase my projects and future plans, including the game: "2 Truths and a Goal."

## 🛠️ Quick Start
npm i
npm run dev  
npm run build  
firebase deploy
