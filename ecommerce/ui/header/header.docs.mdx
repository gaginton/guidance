A header component that renders everything needed for the header including logo and navigation links

### Component usage

```js live
<MemoryRouter>
  <Header
    logoText="Guidance Space"
    src="https://static.bit.dev/bit-logo.svg"
    alt="GuySpace Logo"
  >
    Guidance Space
  </Header>
</MemoryRouter>
```
